# 08 - Demo Seed Dataset

Popular dados realistas para demonstrar o app.

## Acceptance

- [ ] 12 esportes iniciais.
- [ ] 40 perfis de praticantes.
- [ ] 8 professores.
- [ ] 15 aulas.
- [ ] 20 sessoes abertas gratuitas.
- [ ] Convites, participantes e alguns bloqueios/denuncias.

## Notes

Seed deve contar a historia do app sem depender de cadastro manual durante demo.

Dataset pode mostrar professores/aulas com preco e organizadores/entusiastas com plano de assinatura da plataforma quando essa tabela existir. Nenhuma sessao esportiva seedada deve ter preco, taxa, ingresso ou pagamento obrigatorio.
